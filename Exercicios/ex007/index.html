Citações Simples e Completas

Citações simples

Para fazer algumas citações simples usar a tag Q, por exemplo: A vingança nunca é plena, mata a alma e a envenena

Citações Completas

para fazer citações mais completas usas-se a tag BLOCKQUOTE e para linkar basta adicionar CITE ao lado

Códigos

Como Colocar Exemplos de Códigos

Para adicionar alguns exemplos de códigos pode se usar a tag CODE que vai adicionar o código na mesma linha e enquadramento, pórem para deixar com mais detalhes pode-se adicionar a tag PRE antes de code. Vai ficar da seguinte maneira:

    
        num = int(input('digite um número'))
        if num % 2 == 0:
            print(f'0 número {num} é par')
        else:
            print(f'0 número {num} é ímpar')
        print('FIM DO PROGRAMA')

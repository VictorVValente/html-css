Recapitulação Emojis

Para colocar simbolos se coloca & antes de colocar a palavra chave do simbolo e finaliza com ponto e virgula. Exemplo:
&(pound). Que ficaria da seguinte forma:

£

Outro exemplo seria a seta para cima que seria & + uarr + ;
Que ficaria assim:

↑

Para colocar emoji se faz a combinação de & + # + x + o código do emoji
Exemplo:

🤨

Negrito e Italico

Para colocar textos em negirto usar a tag STRONG.
Para colocar textos em itálico usar a tag EM

Marcação

Para deixar textos marcados usar a tag MARK

Textos "Deletados" e "Inseridos (sublinhados)"

Para deixar textos com um risco igual a DELETADO basta usar a tag DEL.
Já para deixar textos INSERIDOS ou SUBLINHADOS basta usar a tag INS

Textos Grandes e Pequenos

Textos Grandes

Para deixar textos,frases e palavras em especifico um pouco maiores, é possivel utilizando a tag BIG porém atualmente esta é um tag obsoleta e não deve mais ser utilizada, mesmo que ainda funcione. O certo agora é estar alterando esse detalhe diretamente em CSS

Textos Pequenos

Para deixar textos,frases ou palavras menores basta utilizar a tag SMALL, esta ainda é uma tag que pode ser utilizada

Textos Sobrescritos e Subscritos

Textos Sobrescritos

Para colocar textos sobrescritos basta usar a tag SUP por exemplo: x3+5

Textos Subescritos

Para colocar textos subscritos usar a tag SUB exemplo: H2O

Citações Simples e Completas

Citações simples

Para fazer algumas citações simples usar a tag Q, por exemplo: A vingança nunca é plena, mata a alma e a envenena

Citações Completas

para fazer citações mais completas usas-se a tag BLOCKQUOTE e para linkar basta adicionar CITE ao lado

Audios

Como reproduzir audios

Basicamente para colocar o audio acima usei a tag AUDIO

Outra maneira de colocar audio que é a mais recomenda e usando a tag AUDIO + SOURCE SRC.

Também se usa a tag CONTROL para adicionar os controles de audio e também e interessante colocar PRELOAD=METADATA para que o site não fique tão pesado.
Se quiser que o audio fiquem tocando sem parar basta colocar LOOP
